### Why Fira

提高视觉渲染效果，提高可读性
![all_ligatures](https://res.cloudinary.com/practicaldev/image/fetch/s--uzH3ViYK--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/tonsky/FiraCode/master/showcases/all_ligatures.png)

### Install on macOS

```shell
$ brew tap caskroom/fonts 
$ brew cask install font-fira-code
```

### enable on VSCode

settings.json
```json
{ 
"editor.fontFamily": "Fira Code", 
"editor.fontLigatures": true
}
```
